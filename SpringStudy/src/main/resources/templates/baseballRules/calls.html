헷갈리는 판정과 심판 제스처를 명확하게 정리했습니다

야구 룰 설명 / 판정 & 심판 신호
TIP 상황 예시와 함께 자주 헷갈리는 판정 위주로 정리했어요
인필드 플라이 (Infield Fly) ☁️

조건: 0~1아웃, 주자 1·2루 또는 만루 상황에서 내야 플라이성 타구 → 심판이 인필드 플라이 선언 시 자동 아웃.

  • 투수, 포수 포함 내야수가 충분히 처리 가능할 때 적용
  • 야수의 포지션이 아니라 타구의 ‘높이·느낌’이 중요
오해 주의: 인필드 플라이는 타자만 아웃이며, 주자는 태그업 후 진루 가능!
체크스윙 (Check Swing) ⚾

정의: 타자가 방망이를 내다 멈추는 애매한 스윙 → 주심이 먼저 판정, 이의 제기 시 루심이 최종 판정.

  • 스윙 여부는 방망이 끝이 홈플레이트를 통과했는지 또는 스윙 의도 유무 기준
  • 루심에게 제스처 요청 시 손으로 원 그리듯 신호
현장 팁: 관중은 헷갈려도 루심 판정은 최종. ⚠️ 비디오 판독 불가 항목 BUT 25년 8월 19일 도입 예정
인터페어런스 (Interference) 🚫

정의: 공격 또는 수비 쪽이 반대편의 플레이를 물리적 또는 시각적으로 방해한 경우 선언.

  • 예: 타자가 캐처의 송구를 막음, 주자가 수비 동작 방해 등
  • 고의성 여부보다 결과적 방해 여부가 핵심
자주 혼동: 수비 방해와 주루방해는 다르다! ⚠️ 공격이 방해 → 아웃 / 수비가 방해 → 진루 허용
그라운드룰 더블 (Ground Rule Double) ⛳

상황: 타구가 바운드 후 외야 펜스를 넘어가면 → 자동 2루타 판정.

  • 야수가 타구를 건드려 바깥으로 튕겨나가도 동일
  • 주자는 당시 위치 고려 없이 2루까지 자동 진루
KBO 특이점: 구장별 펜스 구조 상 자주 나옴. ⚠️ 바운드 여부만으로 HR이 되진 않음.
비디오 판독 (Video Review) 📹

기본: 각 팀은 9이닝 내 2회, 연장전 1회 판독 요청 가능. 요청은 심판에게 제스처로 즉시 전달해야 함.

  • 적용 항목: 홈런 여부 / 세이프-아웃 / 페어-파울 / 포스플레이 등
  • 비적용: 스트라이크/볼, 체크스윙 등은 대상 아님
주의: 요청 기한 넘기면 무효 / 감독만 요청 가능
콜드게임 / 서스펜디드 / 노게임 🌀

콜드게임: 5이닝 이상 진행 후 우천·안개 등으로 경기 중단 → 그 시점까지 기록 반영 후 종료.

서스펜디드: 중단 후 나중에 이어하기로 결정된 경기. 이닝, 주자, 점수 모두 유지.

노게임: 5이닝 미만으로 종료되면 경기 자체가 인정되지 않음. 기록도 모두 무효.

KBO 룰 포인트: KBO는 콜드게임 조건이 MLB보다 다양함 (우천, 조명 고장, 사고 등 포함).
